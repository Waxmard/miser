## miser setup simplefin

Exchange a SimpleFIN setup token for an access URL

### Synopsis

Claim a SimpleFIN setup token and save the access URL to your config.

Get a setup token at https://beta-bridge.simplefin.org/
This token can only be claimed once.

```
miser setup simplefin <setup-token> [flags]
```

  -h, --help   help for simplefin
